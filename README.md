# face-expression-tracker

リアルタイムに表情を検出し、感情を数値化して可視化するブラウザアプリです。  
Webカメラの映像を `face-api.js` で解析し、推定された感情を日本語ラベルと絵文字、バーグラフで表示します。

## 機能概要

- **リアルタイム表情検出**
  - Webカメラの映像から顔を検出
  - 検出された顔の表情を 7 感情に分類
- **感情の日本語表示**
  - 普通（neutral）
  - 幸せ（happy）
  - 悲しい（sad）
  - 怒り（angry）
  - 恐れ（fearful）
  - 嫌悪（disgusted）
  - 驚き（surprised）
- **可視化 UI**
  - その時点でもっとも強い感情を絵文字と大きなテキストで表示
  - 感情ごとのスコアをプログレスバーで表示
  - 顔検出がない場合は「顔を探しています」などのステータスメッセージを表示

## 使用技術

- **HTML / CSS / JavaScript**（バニラ）
- **Tailwind CSS**（CDN 版）  
  UI のスタイリングに使用しています。
- **face-api.js**
  - モデルは GitHub Pages 上の公開モデルを `loadFromUri` で読み込み
  - 使用モデル
    - `tinyFaceDetector`
    - `faceLandmark68Net`
    - `faceRecognitionNet`
    - `faceExpressionNet`

## 使い方

1. このリポジトリ（または `index.html`）を任意のディレクトリに配置します。
2. ブラウザで `index.html` を開きます。
   - 例: ファイルをダブルクリック、またはローカルサーバーから配信
3. ブラウザからカメラアクセスの許可を求められたら **許可** します。
4. 画面中央のカメラプレビューに顔を映すと、右側のパネルに
   - 現在の気分（日本語＋絵文字）
   - 信頼度（%）
   - 各感情のバーグラフ
   がリアルタイムに表示されます。

## 開発メモ

- モデル読み込み完了前は「AIモデルを読み込み中...」というローディングオーバーレイを表示します。
- カメラアクセスが拒否された場合は、画面上にエラーメッセージを表示します。
- メインのロジックは `index.html` 内 `<script>` に実装されています。
